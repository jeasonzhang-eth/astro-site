# 一体化网站配图设计规格

## 目标

为当前“Jeason 个人品牌 + 产品证据 + 倍数引擎公司交付”网站补充少量真正承担解释作用的配图，不把站点改造成图库式企业模板。

## 配图位置

1. 首页首屏：解释个人负责、产品证据和公司交付之间的关系。
2. 服务页开头：解释软件、AI、GEO、自动化和部署是连接的工作流，而不是五个孤立标签。
3. 公司页开头：建立深圳、工程实施和正式公司主体的视觉联系。

项目详情、笔记和联系页不新增插画，避免图片稀释事实与行动信息。

## 视觉系统

- 一张三联画位图作为统一素材，三个页面分别展示不同画幅。
- 无文字、无 Logo、无语言依赖，适配中英文页面。
- 暖象牙白、炭黑、深青绿、珊瑚橙和芥末黄。
- 编辑型技术插画，细墨线、丝网印刷/孔版印刷质感、轻微纸张纹理。
- 避免通用 AI 渐变、发光球体、玻璃卡片、图库人物、握手和夸张机器人。
- 图片承担叙事，正文继续承担事实；所有画面提供双语 alt 与 caption。

## 组件方案

新增 `EditorialArtwork.astro`：

- `variant`: `workbench | systems | company`
- `alt`: 当前语言的替代文本
- `caption`: 当前语言的简短说明
- `priority`: 首页为 true，其他页面懒加载

同一三联画通过 CSS 裁切成三个面板，不生成重复资源。

## 验收标准

- 首页、服务页、公司页均出现对应画幅。
- 中文和英文 alt/caption 正确。
- 图片只有一个网络资源，使用浏览器缓存复用。
- 不产生水平溢出或布局跳动。
- 首页图片预加载优先，其他页面使用 lazy loading。
- 无障碍快照中 figure、image、caption 语义清楚。
